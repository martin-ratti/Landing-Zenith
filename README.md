## 🎯 Sobre el Proyecto

La misión de este sitio es comunicar la propuesta de valor única de **Zenith POS**: un sistema de gestión que no solo controla inventarios y ventas, sino que optimiza la experiencia del operador mediante una interfaz de "Centro de Comando" inspirada en la eficiencia del teclado y la respuesta inmediata.

**Puntos Clave:**
- **Alta Conversión:** Optimizada para la máxima retención y claridad de mensaje.
- **Showcase Interactivo:** Demostración visual de las capacidades del sistema en tiempo real.
- **Responsividad Total:** Diseño adaptado meticulosamente para dispositivos móviles y escritorio.

---

## 🏗️ Arquitectura del Sistema

El proyecto sigue una arquitectura **Component-Based** facilitada por **Astro**, lo que permite un tiempo de carga mínimo mediante la técnica de "Islands Architecture" (aunque actualmente es mayormente estático para máxima velocidad).

### Mapa de Componentes
- `HeroSection`: Impacto visual inmediato y propuesta de valor.
- `InteractiveShowcase`: Puntos calientes (Hotspots) que explican módulos del sistema.
- `FeaturesSection`: Desglose técnico de beneficios.
- `AboutSection`: Historia y narrativa de marca enfocada en el usuario.
- `CustomSoftwareSection`: Formulario de pre-calificación para proyectos a medida.
